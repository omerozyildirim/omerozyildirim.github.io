Online PID Controller Simulation

Below is a brief description of the parameters:

CSTR (Tank) Section

Volume, Height: These define the physical dimensions of the tank.

Inlet, Outlet: These define the maximum capacity of the input and output of the tank.

Inlet%, Outlet%: These represent the opening percentage of the input and output valves respectively. The actual input flowrate is calculated as:

input = inlet * inlet%

The output flowrate is calculated in the same manner.

Level%: This represent the filled percentage of the tank.

KP, KI, KD: These are the proportional, integral and derivative coefficients of the controller. You can activate each mode with the checkboxes.

Process variable: This is the variable that will be controlled, meaning, it will be tried to keep as close as to the setpoint value.

Command variable: This is the variable that will be altered to keep the process variable controlled.

Setpoint: This is the value the process variable will be kept as close to as possible.

Delay: With this parameter you may delay the appliance of the setpoint value.

Run time: This is the total running time of the simulation.

Time step: The default time step value for the simulation is

(Run time) / 10000 sec.

If you want to enter a different value, click the checkbox and enter your own.
