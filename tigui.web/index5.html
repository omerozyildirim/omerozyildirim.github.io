Towhee Input GUI

This is a simple GUI attempt for the software package MCCCS Towhee. The main aim is to ease the parameter writing process by taking the burden of having to remember to include each and every parameter and manually ordering them in the input file off the users' shoulders. Of course, by philosophy a real GUI should free the user from typing in numbers and visualize the information in a fancy and perceptive way and I know this is nothing even near to that. I've tried to cover the all the parameter types as I could except for the fields section and the molecule definitions. They should be included manually to the generated towhee_input file. Any criticisms about the design or any information about the errors are welcome.
